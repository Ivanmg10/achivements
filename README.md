---

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Run the dev server
npm run dev
```

> Make sure to set `NEXTAUTH_URL` and `NEXTAUTH_SECRET` in your `.env.local`.
